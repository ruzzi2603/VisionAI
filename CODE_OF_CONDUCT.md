# Código de Conduta

## Nossa promessa
Como colaboradores e mantenedores deste projeto, nos comprometemos a criar uma comunidade acolhedora, segura e respeitosa para todos.

## Padrões esperados
- Seja cordial e construtivo em todas as interações.
- Respeite pessoas com diferentes opiniões, experiências e identidades.
- Receba feedback com mente aberta e ofereça sugestões de forma educada.
- Respeite a privacidade e dados sensíveis dos outros.

## Comportamento inaceitável
- Assédio, humilhação, discriminação ou ataques pessoais.
- Linguagem abusiva, racista, sexista, homofóbica ou ofensiva.
- Divulgação de dados sensíveis ou informações privadas sem consentimento.
- Spam, trolling ou comportamento perturbador em discussões.

## Aplicação
Se você observar comportamento inadequado, por favor abra uma issue privada ou envie um reporte para os mantenedores do projeto. As contribuições que violarem este código de conduta poderão ser removidas e o autor poderá ser bloqueado.

## Contato
Para reportar abusos ou problemas de conduta, utilize o canal definido pelo mantenedor do repositório ou abra uma issue marcada como `security`/`confidential`.
